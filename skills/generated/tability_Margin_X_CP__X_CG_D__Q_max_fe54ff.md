# 技能文件：火箭初步结构质量估算与气动稳定性快速评估

## 知识要点

### 1. 稳定性余量公式
- **定义**：稳定性余量（Static Margin, SM） = (X_CP - X_CG) / D
  - X_CP：全箭气动压心距参考点距离（通常从鼻锥顶点计）
  - X_CG：全箭质心距参考点距离
  - D：参考直径（通常为箭体最大直径）
- **工程判据**：
  - SM ≥ 1.5 直径：极稳定（业余火箭常用）
  - 1.0 ≤ SM < 1.5：良好稳定
  - SM < 1.0：需精细气动分析，主动控制可行但危险
- **预测方法**：
  - 压心初步估计：使用 Barrowman 方程（修正版）或数值计算
  - 亚音速下简单构型 CP 可近似为各部件面积心按法向力系数加权平均

### 2. 最大动压下结构载荷
- **气动分布载荷**：q_max * A_ref 给出气动合力参考值，单位 N
  - q_max：最大动压（Pa），取决于任务剖面和大气模型
  - A_ref：参考面积，通常取 πD²/4
- **等效轴向力**：需乘以轴向力系数（C_A），粗略估计可取 0.3~0.5（跨音速峰值可能更高）
- **弯曲临界**：当攻角不等于零时，法向力分布产生弯矩，最大弯矩出现在 Q_max * C_Nα * α * A_ref * (X_CP - 某截面)
- **壁厚快速校核**：薄壁圆筒轴向应力 σ = F_axial / (π D t)，需小于材料屈服强度/安全系数

### 3. 快速质量预估方法
- **箭体圆筒段质量**：
  - 公式：m = ρ_material * π * D * L * t
    - 对于给定壁厚 t，直径 D，长度 L，材料密度 ρ
  - 常用材料密度：
    - 6061-T6 铝：2700 kg/m³
    - 304 不锈钢：8000 kg/m³
    - 碳纤维复合材料：约 1600 kg/m³（铺层差异大）
  - 示例：D=0.3m, L=1.5m, t=0.002m, 铝 → m = 2700 * π * 0.3 * 1.5 * 0.002 ≈ 7.63 kg（注意你的计算中 π × 0.15 × 1.5 是周长×长，面积应为 π * D * L * t，D是直径，0.3，不是半径0.15，你的公式有误）
- **其他部件**：鼻锥、尾翼、内结构、发动机舱、电子舱按几何估算或经验因子放大（1.2~1.5倍筒体质量）

### 4. 关键数据补充
- **6061-T6 铝屈服强度**：约 276 MPa（退火态较低，T6态可达 310 MPa），弹性模量 69 GPa
- **大气动压快速参考**（标准大气）：
  - 距地面 0 km：ρ≈1.225 kg/m³
  - 5 km：ρ≈0.736 kg/m³
  - 10 km：ρ≈0.414 kg/m³
  - 预估 Q = 0.5 * ρ * v²
- **典型业余火箭 Max Q**：若最大速度 Ma 0.8~1.2，Max Q 在低空跨音速处，可能 50~200 kPa 量级
