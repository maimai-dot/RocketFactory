## 供应链策略

### 采购原则
- **优先民用替代品**：航天级 → 工业级 → 民用级逐级降级评估
- **分布式采购**：避免单源依赖，每种关键物料至少两个替代供应商
- **批量化**：标准件（螺栓、轴承、O型圈）批量采购，非标件按需定制
- **可追溯性**：所有外购部件保留采购记录、批次号、合格证

### 物料分级
| 等级 | 定义 | 采购策略 | 审批 |
|------|------|----------|------|
| A | 安全关键（推进剂、压力容器、结构件） | 多源认证 + 来料检验 | Safety Officer + Director |
| B | 性能关键（传感器、舵机、电池） | 品牌指定 + 性能测试 | 专业 Chief |
| C | 辅助材料（线缆、紧固件、耗材） | 电商批量采购 | 自由采购 |

## 主要采购渠道

### 1688（大宗批发）
- 金属原材料：不锈钢管/板、铝合金型材、钛合金棒
- 化工原料：硝酸钾、山梨醇、高氯酸铵（需资质）、环氧树脂
- 标准件：高强度螺栓、轴承、密封圈、弹簧

### AliExpress / 速卖通（小批量）
- 电子模块：IMU 传感器、GPS 模块、STM32 开发板、数传模块
- 舵机/电机：大扭力金属齿轮舵机、无刷电机、电调
- 碳纤维制品：碳纤维管、碳纤维板、预浸料

### 淘宝（零散采购）
- 实验室器材：烧杯、量筒、温度计、真空泵
- 工具：螺丝刀套装、焊接工具、热风枪、3D 打印机耗材
- 安全防护：防护眼镜、耐高温手套、防毒面具

## 成本估算模型

### 1 公里级验证箭 BOM 估算
| 系统 | 物料 | 估算成本(RMB) |
|------|------|---------------|
| 箭体结构 | 不锈钢管材 + CNC 加工 | 2000-5000 |
| 发动机 | KNSB 固体 / 小型液体 | 1000-3000 |
| 飞控硬件 | STM32 + IMU + GPS + 数传 | 300-800 |
| 舵机/执行器 | 4× 大扭力舵机 + TVC 机构 | 200-600 |
| 回收系统 | 降落伞 + 释放机构 | 200-500 |
| 地面站 | 笔记本 + 数传模块 + 发射架 | 500-1500 |
| 耗材 | 3D 打印耗材、焊材、胶水 | 500-1000 |
| **合计** | | **4700-12900** |

### 降本策略
1. 3D 打印非承力结构件（PLA/PETG 替代铝合金）
2. 使用标准型材减少 CNC 加工时数
3. 淘宝二手/拆机传感器（校准后使用）
4. 社区借力：开源硬件社区获取设计文件

## BOM 清单管理

### BOM 条目格式
```
[编号] | [名称] | [规格型号] | [数量] | [单价] | [供应商] | [链接] | [交期] | [等级] | [备注]
```

### 版本管理
- 每次设计变更同步更新 BOM
- 标注变更原因和日期
- 保留历史版本供审计

## 物流与仓储

### 危险品管理
- 推进剂原料：远离热源/火源，独立上锁存储
- 氧化剂与燃料物理隔离存放
- 量不超过试验所需最小量的 2 倍

### 物流时间线
- 国内电商：3-7 天
- 1688 非标定制：1-4 周
- AliExpress 跨境：2-6 周
- 物流时间纳入项目甘特图
