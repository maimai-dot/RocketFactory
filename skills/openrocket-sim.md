## OpenRocket 基础

### 软件能力边界
- **可处理**：亚音速到低超音速（Ma < 3），常规箭体构型，标准大气
- **不可处理**：高超声速气动加热、非轴对称外形、复杂级间分离
- **精度**：CP 位置误差 ~5-10%，适用于概念设计和初步设计阶段

### 坐标系
- +X 沿箭体纵轴向前（从尾部指向鼻锥）← 注意：与 FreeCAD 约定相反！
- +Y 为侧向，+Z 为俯仰方向
- 原点位于箭体顶点（鼻锥尖端）

## 建模工作流

### 1. 箭体搭建
- 从鼻锥开始：选择型线（von Kármán / 圆锥 / 抛物线 / 椭圆）
- 添加箭体管段（Body Tube）：设置直径、长度、壁厚、材料
- 添加过渡段（Transition）：锥形或曲线过渡
- 添加发动机舱（Inner Tube）：发动机安装位置

### 2. 气动面
- 弹翼（Fins）：选择翼型（梯形、三角形、自定义），设置根弦、梢弦、展长、后掠角
- 栅格翼：使用多片平行弹翼近似
- 着陆腿（展开/收起两种构型分别建组）

### 3. 质量组件
- 有效载荷质量（Mass Component）
- 降落伞（Parachute）：含展开条件（高度、时间）
- 发动机（Motor）：从推力曲线数据库选择或自定义

### 4. 仿真配置
- 发射条件：发射角度、风速/风向、发射导轨长度、海拔
- 仿真选项：大气模型（ISA）、地球模型（WGS84 椭球 / 平面）、时间步长
- 输出变量：高度、速度、加速度、马赫数、动压、稳定性、姿态

## 稳定性分析

### 核心判据：静稳定裕度
```
Stability Margin = (X_CP - X_CG) / D
```
- X_CP：压心位置（气动合力作用点）
- X_CG：重心位置
- D：箭体参考直径
- **要求**：全飞行过程 ≥ 1.5 cal（口径），亚音速段 ≥ 2.0 cal 更安全

### 动态稳定性
- 阻尼比：俯仰/偏航振荡衰减特性
- 气动阻尼系数：由箭体几何和飞行速度决定
- 若出现欠阻尼振荡，需增大弹翼面积或改变布局

## CLI 批量仿真

### 命令行调用模式
```bash
# 单次仿真
java -jar OpenRocket.jar --sim <sim_id> <ork_file> --output <csv_file>

# 参数扫描（通过 Python 封装）
python openrocket_tools.py sweep --param body_diameter --range 0.1 0.3 0.02
```

### Python 封装要点
- 调用 OpenRocket JAR 文件的命令行接口
- 解析输出 CSV 提取关键指标（最大高度、最大速度、稳定裕度曲线）
- 使用 matplotlib 生成仿真报告图表
- 支持并行参数扫描（不同参数组合分发到多进程）

## 仿真数据后处理

### 必提取指标
- 最大飞行高度（Apogee）
- 最大速度（Vmax）和最大马赫数（Ma_max）
- 最大动压（Q_max）——结构设计载荷
- 稳定裕度曲线（Caliber vs time）
- 着陆点散布（含风场扰动）

### 报告图表模板
1. 高度-时间曲线
2. 速度-时间曲线
3. 加速度-时间曲线
4. 动压-时间曲线
5. 稳定裕度-时间曲线（标注 ±1.5 cal 安全区间）
6. 弹道三维轨迹图
