## 三、Skills（技能）如何创建与使用

技能是 `.md` 文件，存放在 `skills/` 目录下。它们会被自动发现并注入到对应智能体的“知识大脑”中，指导其如何专业地思考问题。

### 创建技能文件的通用模板

在 `skills/` 下新建 `freecad-rocket.md`，内容如下（其他技能类似）：

```markdown
---
name: freecad-rocket-design
description: FreeCAD火箭结构设计专业知识与操作流程
---

## 火箭结构设计原则
- 箭体直径需根据发动机推力级和稳定性要求确定，通常采用 0.15-0.3m 直径的缩比箭体。
- 重心（CG）必须位于压心（CP）之前至少 1.5 倍箭体直径，以保证被动稳定性。
- 鼻锥建议采用 von Kármán 曲线，减少跨音速阻力。
- 栅格翼和着陆腿需设计为可折叠机构，使用铰链和弹簧驱动，收起后贴合箭体外壁。

## FreeCAD 操作要点
- 使用 Part Design 工作台创建箭体分段实体。
- 通过 Spreadsheet 工作台管理参数（直径、壁厚、材料密度），确保全箭参数化。
- 完成设计后，使用 `Mesh Workbench` 导出 STL 文件，供 3D 打印。
- 使用 Python API 提取质量、重心和转动惯量，输出为 JSON。

## 迭代流程
1. 根据 Director 下达的总体参数，创建初始箭体模型。
2. 提取质量特性，交给 Sim Chief 进行飞行仿真。
3. 根据仿真结果调整结构（如减小重量、改变翼面积）。
4. 重新仿真，直至达到性能目标。
